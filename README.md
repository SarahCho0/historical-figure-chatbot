## 🚀 Getting Started

### Prerequisites

- An **OpenAI API Key** with `gpt-4o` access → [Get one here](https://platform.openai.com/api-keys)
- A modern browser (Chrome, Safari, Firefox, Edge)
- Internet connection (for OpenAI API + Google Fonts)

### Running the App

```bash
# macOS — open directly in your default browser
open historical-figures-chatbot.html

# Or target a specific browser
open -a "Google Chrome" historical-figures-chatbot.html
open -a "Safari" historical-figures-chatbot.html
```

> You can also simply **double-click** `historical-figures-chatbot.html` in Finder.

## 🕹️ How to Use

```
① Enter your OpenAI API Key in the sidebar
         ↓
② (Optional) Switch language: EN ↔ 한국어 using the toggle in the top-right
         ↓
③ Select a historical figure from the list
         ↓
④ The entire UI transforms to match the era & personality
         ↓
⑤ Choose a chat mode (General / Deep Reflection / Era Context / Teach Me)
         ↓
⑥ Ask anything — use example chips or type your own question
```

> If a response is cut off, a **📖 Continue Reading** button appears at the bottom of the message. Click it to receive the rest seamlessly.

## 🔒 Security Notes

> ⚠️ This app was built for **local use only**.

- Your API Key is held **in-memory only** and is never written to disk or sent anywhere except OpenAI's API.
- Do **not** hardcode your API Key into the source file.
- Do **not** upload this file to a public repository with your API Key inside.
- If you deploy this to a web server, implement a proper backend proxy to protect your key.

## ❓ Troubleshooting

| Symptom | Fix |
|---|---|
| Input box is disabled | Select a historical figure from the sidebar first |
| `API Error: 401` | Invalid or expired API Key — double-check and re-enter |
| `API Error: 429` | Rate limit exceeded — wait a moment and try again |
| Mode tabs not visible | Tabs appear automatically after selecting a character |
| Fonts rendering as default | Check your internet connection (Google Fonts must load) |
| Response cut off mid-sentence | Click the **📖 Continue Reading** button below the message |
| Background isn't changing | Update to the latest version of Chrome, Safari, or Firefox |
